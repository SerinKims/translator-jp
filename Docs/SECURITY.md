# SECURITY & DATA POLICY

## 1. 데이터 보관 원칙

```text
사용자의 원문과 번역문은 로컬 DB에 저장한다.
pixiv 원문은 사용자의 번역 목적 범위에서만 저장한다.
외부 API로 원문을 전송하지 않는다.
로그에는 전체 원문을 남기지 않는다.
API 키, 인증 정보, 환경 변수는 DB에 저장하지 않는다.
pixiv 로그인 정보는 DB에 저장하지 않는다.
긴 소설 원문은 저작권 이슈가 있을 수 있으므로 외부 전송을 금지한다.
수집한 원문을 재배포하지 않는다.
raw_model_response 저장 여부는 환경 변수로 제어한다.
```

---

## 2. pixiv 관련 금지 사항

```text
pixiv 대량 크롤링 금지
태그, 랭킹, 작가 페이지 순회 수집 금지
로그인, 결제, 연령 제한, 권한 제한 우회 금지
캡차 우회 금지
robots.txt 또는 접근 제어 우회 금지
비정상적인 고속 요청 금지
수집 원문 재배포 금지
수집 원문을 학습 데이터셋으로 무단 사용 금지
```

---

## 3. 외부 전송 금지

다음 데이터는 외부 API로 전송하지 않는다.

```text
pixiv 원문 전체
사용자 입력 원문 전체
번역 결과 전체
사용자 수정/피드백 원문 전체
```

이 프로젝트의 기본 LLM Runtime은 로컬 LiteRT-LM이다.

---

## 4. 인증 정보 관리

```text
환경 변수는 .env 또는 OS 환경 변수로 관리한다.
환경 변수 값을 DB에 저장하지 않는다.
pixiv 로그인 정보는 서버에 저장하지 않는다.
세션 정보 저장 기능을 구현할 경우 로컬 전용이며 별도 ADR이 필요하다.
```
